# HTTP

HTTP-коды ответа T-Invest API.

| <nobr>Код ошибки</nobr> | Тип ошибки                            | Текст ошибки           | Причины возникновения и <br/> рекомендации по устранению                                                                            |
|------------|---------------------------------------|------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| 200        | OK                                    |                        | Успешный запрос                                                                                                                     |
| 400        | INVALID_ARGUMENT                      | Bad Request            | Запрос составлен неверно, проверьте параметры вызова и синтаксические ошибки.                                                       |
| 401        | UNAUTHENTICATED                       | Unauthorized           | Токен доступа не найден или не активен.<br/>Новый токен можно выпустить в [личном кабинете](https://www.tbank.ru/invest/settings/). |
| 403        | PERMISSION_DENIED                     | Forbidden              | Недостаточно прав для выполнения запроса. Проверьте соответствие токена и передаваемого в запросе счета.                            |
| 404        | Unimplemented                         | Not Found              | Неверный URL запроса.                                                                                                               |
| 405        |                                       | Method Not Allowed     | Вызов метода недоступен. Попробуйте повторить запрос позже.                                                                         |
| 408        |                                       | Request Timeout        | Превышено время ожидания ответа от сервера. Попробуйте повторить запрос позже.                                                      |
| 415        | Internal                              | Unsupported Media Type | Неверный тип данных.                                                                                                                |
| 426        |                                       | Upgrade Required       | Обновите версию протокола в вызове.                                                                                                 |
| 429        | RESOURCE_EXHAUSTED                    | Too Many Requests      | Превышен лимит запросов в минуту. [Подробнее про лимитную политику](/invest/intro/intro/limits/).                                   |
| 500        | INTERNAL, UNKNOWN, DATA_LOSS, Unknown | Internal Server Error  | Внутренняя ошибка сервера.                                                                                                          |
| 501        | UNIMPLEMENTED                         | Not Implemented        | Сервер не поддерживает функцию, нужную для выполнения запроса.                                                                      |
| 503        | UNAVAILABLE                           | Service Unavailable    | Сервис временно недоступен. Попробуйте повторить запрос позже.                                                                      |
| 504        | DEADLINE_EXCEEDED                     | Gateway Timeout        | Нет ответа от шлюза или прокси-сервера.                                                                                             |
