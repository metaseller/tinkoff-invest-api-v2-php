# Особенности торговли на внутриброкерском рынке

В Т-Инвестициях реализовано несколько дополнительных внебиржевых режимов торгов, которые расширяют биржевую торговлю:

* [Дилерские торговые сессии](/invest/intro/useful-info/weekend-trading/extended) — торги **биржевыми** инструментами 
в выходные и нерабочее время.

* [Инструменты, по которым приостановлена биржевая торговля](/invest/intro/useful-info/weekend-trading/held_instruments) — операции с инструментами
  иностранных компаний.

* [Фонды Т-Банка](/invest/intro/useful-info/weekend-trading/tbank_fund) — торги фондами Т-Банка.

## Признаки инструментов
| Режим торгов                      | Особенности                                                                                                                                                                                         |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Дилерские торговые сессии         | Инструменты, в [расписании](/invest/services/instruments/head-instruments#получить-расписание-торгов) торговой площадки которых есть интервал `dealer_regular_trading_session`.                     |
| Инструменты иностранных эмитентов | Акции и фонды [торговой площадки](/invest/intro/useful-info/markets#торговые-площадки) `dealer_int_exchange`.                                                                                       |
| <nobr>Внебиржевые фонды Т-Банка</nobr>         | Инструменты [торговой площадки](/invest/intro/useful-info/markets) `dealer_realestate` и [class_code](/invest/services/instruments/faq_instruments#что-такое-class_code-и-где-его-найти) `ZPIF_OTC`. |
