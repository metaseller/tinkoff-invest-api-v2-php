# Конкурсные работы

Мы проводили конкурсы по разработке торгового робота для Т-Инвестиций. Задача участников — написать пример торговой системы, которая реализует любую торговую стратегию на бирже и работает через новый API Т-Инвестиций. 

Ниже собраны работы сообщества, которые были сделаны в рамках конкурсов. Работы созданы энтузиастами — используйте их на свой страх и риск.
