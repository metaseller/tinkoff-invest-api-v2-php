# Примеры роботов и интеграций

В зависимости от опыта и навыков разработки, можно выбрать один из вариантов взаимодействия с T-Invest API:

- Использовать готовое решение.
- Разработать собственного робота на одном из языков программирования.
- Использовать LLM.

## Готовые платформы

Существуют готовые платформы для алготрейдинга, которые интегрированы с T-Invest API. Они предоставляют дружелюбный интерфейс и не требуют серьезных навыков программирования. Примеры плафторм:

- [OSEngine](https://o-s-a.net/os-engine.html)
- [Stocksharp](https://stocksharp.ru/)
- [TSLab](https://www.tslab.pro/)
