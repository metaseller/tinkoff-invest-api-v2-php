# Переводы между счетами

## Описание

Сервис счетов позволяет получить список дебетовых счетов и информацию о балансе, пополнить брокерский счет, а также переводить денежные средства между брокерскими счетами.
Вывод средств с брокерского счета недоступен.

## Токен

Чтобы воспользоваться методами для перевода средств между счетами, в запросе нужно передать 
токен с правами на переводы.

Токен T-Invest API можно выпустить на странице [настроек профиля Т-Инвестиции](https://www.tbank.ru/invest/settings/) — выберите доступ 
«Торговля и переводы». Так как для перевода требуется доступ к двум счетам — списания и зачисления, 
выпущенный токен может быть только с правами на все счета.

[Подробнее о токенах](/invest/intro/intro/token)

## Методы

С помощью методов сервиса можно:
- получить [список дебетовых и накопительных счетов](/invest/services/accounts/users#getbankaccounts) и доступных остатков на них;
- сделать [перевод денежных средств между брокерскими](/invest/services/accounts/users#currencytransfer);
- [пополнить брокерский счет](/invest/services/accounts/users#payin) с карты Т-Банка или накопительного счета;


## Ограничения на переводы денежных средств

- Переводы можно выполнять только между брокерскими счетами.
- Пополнения и списания со счетов ИИС, «Инвесткопилки» и «Счёта под ключ» недоступны.
- Переводы доступны в одной из валют: `rub`, `usd`, `eur`, `cad`, `chf`, `gbp`, `hkd`, `sek`, `try`, `cny`, `jpy`, `kzt`, `byn`, `amd`, `uzs`, `kgs`, `tjs`.
- Пополнения доступны только в рублях.
- Минимальная сумма перевода — 0.01 в валюте перевода.
- Сумма перевода или пополнения не должна превышать доступный остаток на счете списания.
